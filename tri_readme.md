触发器文档
=============

RS
---------
### 输入
* 0 : R
* 1 : S
### 输出
* 0 : Q
* 1 : !Q

RSC（带时钟信号的RS）
----------
### 输入
* 0 : R
* 1 : S
* 2 : CP
### 输出
* 0 : Q
* 1 : !Q

D
---------
### 输入
* 0 : D
* 1 : CP
### 输出
* 0 : Q

JK
----------
### 输入
* 0 : J
* 1 : K
* 2 : CP
### 输出
* 0 : Q
* 1 : !Q

TT
--------
### 输入
* 0 : T
* 1 : CP
### 输出
* 0 : Q
